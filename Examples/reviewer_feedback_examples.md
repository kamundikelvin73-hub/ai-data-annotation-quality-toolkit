# Reviewer Feedback Examples

## Purpose

Reviewer feedback should help annotators improve quality and consistency.

## Feedback Template

```text
Issue:
Correct label:
Reason:
Guideline reference:
Recommended action:
```

## Example 1

Issue:
The text was labeled Technical Support.

Correct label:
Account Access

Reason:
The user is unable to log into the account. Login and authentication issues belong under Account Access.

Guideline reference:
Text Classification Guidelines > Label Taxonomy

Recommended action:
Review the Account Access definition and relabel similar examples.

## Example 2

Issue:
The sentiment was labeled Neutral.

Correct label:
Negative

Reason:
The user explicitly says they are frustrated and disappointed.

Guideline reference:
Sentiment Annotation Guidelines > Negative label definition

Recommended action:
Use expressed emotion as the main signal for sentiment.

## Example 3

Issue:
Entity span is incomplete.

Correct label:
"University of St. Thomas" as ORGANIZATION

Reason:
The full organization name should be labeled.

Recommended action:
Review full-span rules in the entity labeling guideline.
